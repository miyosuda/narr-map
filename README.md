# narr-map

A minimal mind map editor made with Electron. The interface is almost compatible with Free-mind.

## Shortcuts

| Key |     |
| ---- | ----  |
| Enter   | Add sibling node |
| Tab     | Add child node |
| Space   | Toggle fold   |
| Ctrl+i  | Edit text  |
| Ctrl+n  | Move down  |
| Ctrl+p  | Move up  |
| Ctrl+f  | Move right |
| Ctrl+b  | Move left |
| F2      | Edit text  |



## Development

### Initial setup

```
$ npm install
```


### Debug

```
$ npm start
```


### Build

```
$ npm run make
```

### Test

```
$ npm run test
```
