# narr-map

A minimal mind map editor.


## Icons

| text | Icon      |
| ---- | ----  |
| (r)  | :red_circle:   |
| (g)  | :green_circle:  |

| (y)  | :yellow_circle:  |


## Development

### Initial setup

```
$ npm install
```


### Debug

```
$ npm start
```


### Build

```
$ npm run make
```
