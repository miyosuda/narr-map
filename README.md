# narr-map

A minimal mind map editor.



## Icons

| Text | Icon    |
| ---- | ----  |
| (r)  | :red_circle:   |
| (g)  | :green_circle:  |
| (b)  | :large_blue_circle:   |
| (y)  | :yellow_circle:  |



## Shortcuts

| Key |     |
| ---- | ----  |
| SPACE  | Toggle fold   |
| Ctrl+i  | Edit text  |
| Ctrl+n  | Move down  |
| Ctrl+p  | Move up  |
| Ctrl+f  | Move right |
| Ctrl+b  | Move left |



## Development

### Initial setup

```
$ npm install
```


### Debug

```
$ npm start
```


### Build

```
$ npm run make
```
